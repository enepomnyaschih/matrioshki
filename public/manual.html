Matreshki vs Androids

History

Androids got out of control and captured all GDD hosting locations excepting Moscow. And only Russian matreshki can fix androids up and safe the feast!

Goals of the game

  1. Secure all GDD hosting locations
  2. Don't let Androids to capture all GDD hosting locations

Game rules

  1. 2 players participate in the game: human vs computer
  2. Human controls matreshki
  3. Computer controls androids
  4. Game map is split to 25 areas
  5. Each area is under control of some player initially:
    1. Moscow area is under control of matreshki
    2. Other GDD hosting areas are under control of androids
    3. Remaining areas are arranged among players randomly in 3:1 (androids:matreshki) relation
  6. There is a unit in each area: matreshka or android
  7. Units have different sizes: there are 8 levels of unit size
  8. Human player makes first turn
  9. While player has turn on, he can make any number of attacks
  10. To attack, player must click source area and then click target area:
    1. Source and target areas must be adjacent
    2. Source area must belong to current player
    3. Target area must belong to enemy player
    4. Source area must have unit of at least 2'nd level
  11. Battle happens automatically in the next order:
    1. Each player is rolling the dices
    2. Number of dices equals to level of corresponding unit
    3. Dices' points are summed
    4. If attacking player has more points, then he wins the battle:
      1. Attacking player captures the area
      2. Attacking unit's level is decreased by 1 and he moves to captured area
      3. Attacking area gets the unit of 1'st level
    5. Else attacking player retreats:
      1. Attacking area's unit is weakened to 1'st level
  12. Player can click "End turn" button if he doesn't want to attack anymore:
    1. Current player gets reinforcements:
      1. The maximum number of joined areas under player's control is calculated
      2. Random units under player's control grow by the same number of levels
      3. No units grow over 8'th level
    2. Another player starts his turn
  13. If player secures all GDD hosting areas, he wins the game

How we developed this game

The game is developed for participation in Google Open Call for Google Developer Day 2011 Challenge - Doodle the Dymaxion Map.

Implementation notes

The application source source code consists of several JS-classes which manupulate page SVG-content. Source code is open for access on https://github.com/enepomnyaschih/matrioshki. The application is based on next frameworks:

  1. jQuery for simple DOM-manipulations
  2. Raphaël for graphics primitives output
  3. jWidget - author's library, providing rich JS-utilities set

The main issue which obstructed development - creating of SVG-elements based on existing SVG-resources from JS-code. Developers were forced to convert all SVG-resources from XML-format to lots of Raphaël-calls.
